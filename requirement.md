# Project Requirements: Premium Property Agent Website

## 1. Overview
A high-end, premium company profile website for a Property Agent designed to showcase luxury listings and professional services. The site focuses on visual excellence and smooth user experience.

## 2. Tech Stack
- **Framework**: Next.js 14+ (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Runtime**: Node.js

## 3. Key Features
- **Hero Section**: Dynamic search and high-quality visuals.
- **Property Listing**: Grid view with property details (Price, Beds, Baths, Area).
- **Service Highlights**: Highlighting unique selling points (Expert Consulting, Market Analysis, etc.).
- **Responsive Design**: Seamless experience across Mobile, Tablet, and Desktop.
- **Glassmorphism UI**: Modern transparent effects for a premium feel.
- **Smooth Transitions**: Scroll-based animations and hover effects.

## 4. Visual Identity
- **Primary Colors**: Deep Navy (#0F172A), Gold Accent (#D4AF37), Clean White (#FFFFFF).
- **Typography**: Playfair Display (Serif) for headings, Inter (Sans) for body text.
- **Imagery**: High-resolution architectural and interior photography.

## 5. Deployment Goal
- Optimized for static hosting (GitHub Pages / Netlify).
